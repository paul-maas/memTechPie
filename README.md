# memTechPie — Memory Tech KB (mono-vault repo)

Многослойная база знаний для коучинговой/менторской/тренерской практики Pavel'я. AI-операемая, потребляется через [obsidian-intelligence MCP](https://github.com/paul-maas/obsidian-intelligence) (версия >= 0.2.2).

## Три vault'а в одном репо

Каждая подпапка верхнего уровня — отдельный Obsidian-vault с собственной коллекцией ChromaDB:

| Vault | Назначение |
|---|---|
| `kb-knowledge/` | Конституция метода, онтология школ, атомарные карточки концепций/техник/фреймворков, источники, плейбуки оператора |
| `kb-clients/` | Кейсы клиентов, эпизодическая память по сессиям, плейбуки работы с клиентом |
| `kb-ops/` | Audit log AI-операций, отчёты о здоровье базы, метрики, inbox изменений на review |

Изоляция на уровне коллекций Chroma — клиентский слой не смешивается с доменным.

## Архитектура

Полный документ концепции и спецификация расширений MCP лежат на уровень выше, в `Memory Tech Project/`:

- `../00_Архитектура_базы_знаний.md` — основная архитектура (актуальная версия в `version:` frontmatter)
- `../02_Спецификация_расширений_MCP.md` — спецификация MCP-расширений 0.2.2

Эти файлы не версионируются в этом репо намеренно — они принадлежат проектному уровню, а не vault'ам.
