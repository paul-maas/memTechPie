# kb-ops

Операционный vault: audit log AI-действий, отчёты о здоровье базы, метрики, inbox pending-изменений.

## Структура папок

```
audit/             — append-only лог AI-операций над kb-knowledge и kb-clients (по дням: YYYY-MM-DD.md)
reports/           — ежемесячные отчёты о здоровье базы (YYYY-MM.md)
metrics/           — снимки метрик (carded counts, soft-limit warnings, средний возраст canonical, и т.п.)
inbox/             — DECISIONS на review (промоции/мерджи/конфликты/новые узлы онтологии). Блокирует связанное действие AI до ответа. Не путать с kb-knowledge/inbox/ (digest)
```

## Принципы

- **Audit append-only.** AI добавляет записи, не правит существующие. Откат — только через явный rollback-плейбук.
- **Метрики собираются автоматически** через скрипты в `../scripts/`; `vault_stats` MCP — один из источников.
- **Inbox разгребается Pavel'ем еженедельно** (ритуал). Каждая запись содержит обоснование AI, ссылки на источники, ожидаемое действие.
