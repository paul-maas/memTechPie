# kb-knowledge/inbox/ — DIGEST

Не путать с `kb-ops/inbox/`. Здесь — **дайджесты AI-активности** на домене: сводки за сессию ингеста, за день, за неделю. Не блокирует AI, читается на досуге.

## Роль

«Что я (AI) сделал на домене?» Один файл = одна сессия/день.

- `YYYY-MM-DD-<session-id>.md` — сводка сессии: что прочитал, какие draft-карточки создал, какие потенциальные дубликаты нашёл, какие конфликты помечены, какие пробелы онтологии эскалированы.
- Каждая запись содержит ссылки на конкретные карточки в `cards/` и записи в `kb-ops/inbox/` (если есть pending decisions).

## Не путать

| | `kb-knowledge/inbox/` (этот) | `kb-ops/inbox/` |
|---|---|---|
| Роль | **digest** — что сделал AI | **decisions** — что решить Pavel'ю |
| Блокирует AI? | нет | да (для связанного действия) |
| Частота | одна запись = одна сессия/день | одна запись = одно решение |
| Шаблон | `digest-entry.md` | `review-entry.md` |
